# ANIVA
推しが実在する世界。

アニメキャラクターのOnlyFans × デジタルツイン プラットフォーム。

## Tech Stack
- Web: Next.js 15, TypeScript, Tailwind CSS, PWA
- Mobile: React Native (Expo)
- Backend: Next.js API Routes, tRPC
- AI: Claude API, GPT-SoVITS, Flux+LoRA, Live2D
- DB: PostgreSQL, Redis, pgvector
- Payment: Stripe
