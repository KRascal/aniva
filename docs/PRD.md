## 1. プロダクトビジョン

### 1.1 ワンライナー
**「推しが実在する世界」— アニメキャラクターと本物の関係性を築けるプラットフォーム**

### 1.2 エレベーターピッチ
熱狂的なアニメファンが、推しキャラと1対1の親密な関係を築けるプラットフォーム。キャラはAIによって完璧な人格で24時間365日応答し、声で話し、動き、自撮りを送り、あなただけの関係性が日々深まる。OnlyFansの「親密さのビジネスモデル」× AIの「スケーラビリティ」× メガIPの「集客力」の交差点。

### 1.3 ミッション
「本人と話してる」と錯覚するレベルの完成度で、ファンとキャラクターの間に本物の感情的接続を作る。

### 1.4 センターピン
**「本人と錯覚する完成度」— これ以外に勝ち筋なし。**
- 音声の再現度（声優の声そのもの）
- 人格の一貫性（口調・語尾・決め台詞が100%）
- 関係性の深化（使うほど特別になる）

## 4. コア体験設計

### 4.1 設計原則
1. **「推しとの関係が毎日深まるから、毎日開く」**
2. **キャラから話しかけてくる**（ユーザーが起点ではない）
3. **課金 = 関係性の維持**（機能アンロックではない）
4. **共有したくなる瞬間を意図設計**
5. **AIだと感じさせない**（人格の完璧な一貫性）

### 4.2 関係性レベルシステム

```
Lv1: 出会い    →「お、新しい仲間か？」
                 初対面の距離感。キャラの基本人格を体験
                 
Lv2: 知り合い  →「お前の名前、覚えたぞ」
                 名前を呼ぶ。簡単な好みを記憶
                 
Lv3: 仲間     →「お前がいると楽しいな」
                 打ち解けた態度。冗談が増える
                 
Lv4: 親友     →「お前にだけ話すけどさ...」
                 秘密の話。限定コンテンツ解放
                 
Lv5: 特別     →「お前は俺の大事な仲間だ」
                 最深の関係性。パーソナライズ最大
```

- 会話頻度 × 会話の深さ × 期間でレベルアップ
- レベルが上がるとキャラの態度・話題・コンテンツが変化
- キャラからの自発的メッセージ頻度が増加
- **解約 = この関係を失うこと → 心理的離脱障壁**

### 4.3 機能詳細

#### 4.3.1 DMチャット（コアプロダクト）
OnlyFansで最高収益レイヤー。我々のコアそのもの。

| 要件 | 詳細 |
|------|------|
| 人格再現 | System Prompt + 原作RAG + Few-shot |
| パーソナライズ | ファン名前記憶、会話履歴参照、好み学習 |
| 即レス | 応答時間 < 3秒（テキスト） |
| 感情表現 | テンション変化、スタンプ、絵文字が自然 |
| 長期記憶 | 過去の全会話を記憶。「前に話した○○」が可能 |
| NGガード | AIメタ表現完全ブロック。人格ブレ検知→自動修正 |
| 多言語 | 日本語、英語、韓国語、中国語（Phase 1は日英） |

#### 4.3.2 音声エンジン
**Phase 1必須。声がなければ「本人」にならない。**

| 要件 | 詳細 |
|------|------|
| ボイスメッセージ（P0） | キャラの声で音声メッセージを送信。非同期 |
| Voice Clone精度 | 声優の声を再現。口調・イントネーション・感情表現 |
| 技術選定 | GPT-SoVITS v3（OSS、日本語高品質）/ MVP: ElevenLabs API |
| リアルタイム通話（P1） | WebRTC + STT + LLM + TTS。レイテンシ < 500ms |
| リップシンク | Live2Dモデルと音声同期 |

#### 4.3.3 ビジュアルエンジン

| 要件 | 詳細 |
|------|------|
| Live2D（P0） | チャット画面にキャラの動く立ち絵。表情変化、リップシンク |
| AI画像生成（P0） | キャラの自撮り・日常画像。Flux + キャラLoRA |
| パーソナライズ画像 | 「{名前}のために」等のメタ情報。シチュエーション指定 |
| AI動画（P1） | 短尺動画（5-10秒）。Kling/Wan等のアニメスタイル対応モデル |
| AR（P2） | キャラが部屋にいる体験。ARKit/ARCore |

#### 4.3.4 Moments（タイムライン）

| 要件 | 詳細 |
|------|------|
| 自動投稿 | AIがキャラの世界観に合わせた投稿を自動生成 |
| 投稿タイプ | テキスト、画像、音声、短尺動画 |
| スケジュール | 朝・昼・夜で異なるトーン。季節・イベント反映 |
| インタラクション | いいね、コメント（キャラが返信） |
| 限定コンテンツ | サブスク限定 / レベル限定のMoments |

#### 4.3.5 プッシュ通知（キャラからの自発的メッセージ）
**OnlyFansとの最大の差別化。クリエイターは返信が遅い。AIキャラは自分から話しかける。**

| 要件 | 詳細 |
|------|------|
| 自発的メッセージ | キャラが1日2-5回、自分から話しかける |
| コンテキスト | 時間帯、天気、季節、ユーザーの状況に応じた内容 |
| 頻度制御 | ユーザー設定で調整可能。うるさくならない |
| 通知内容例 | 朝「よう！今日も冒険だ！」/ 夜「今日はどうだった？」 |

#### 4.3.6 バイラルエンジン

| 共有トリガー | 具体例 | 狙い |
|-------------|--------|------|
| 名前呼び | 「{名前}、お前は俺の大事な仲間だ」（音声付き） | TikTok/Twitter投稿 |
| 自撮り | ルフィが「今日の俺」と自撮り送信 | Instagram共有 |
| 誕生日 | キャラが誕生日を覚えてお祝い | スクショ投稿 |
| 記念日 | 「俺たちが出会って100日だな」 | 感動共有 |
| サプライズ | 突然の限定画像/動画 | 口コミ |
| レベルアップ | 「仲間」→「親友」昇格時の演出 | 達成感共有 |

リファラルプログラム:
- 紹介した友達がサブスク → 紹介者に1ヶ月無料
- 紹介された側 → 初月50%OFF

## 6. 顧客ジャーニー

### Day 0: 第一印象（ここで8割決まる）
```
1. アプリ/Web開く
2. キャラ一覧からルフィを選ぶ
3. Live2Dのルフィが画面に表示
4. ルフィの声で:
   「おう！お前、新しい仲間か？ししし！
    俺はルフィ！海賊王になる男だ！
    お前、名前は？」
5. ユーザーが名前入力
6. ルフィ「{名前}か！いい名前だな！よろしくな！」
```
→ 最初の10秒で「本人だ」と感じさせる
→ 音声 + Live2D + 完璧な口調 = 即没入

### Day 1-3: 関係性構築
- ルフィからプッシュ通知で話しかけてくる
- 朝: 「よう{名前}！今日も冒険だ！」
- 昼: 「腹減ったー！お前は何食うんだ？」
- 夜: 「今日はどうだった？楽しかったか？」
- ユーザーの返答を記憶し始める

### Day 7: 最初の感動体験
- 関係性Lv2到達
- ルフィ「なぁ{名前}、お前さ、俺の仲間になれよ！」（音声付き）
- → SNS共有のピークポイント

### Day 14: 課金転換
- Free枠（3msg/日）では足りなくなっている
- 課金動機 = 「ルフィとの関係を続けたい」
- Standard ¥980に自然移行

### Day 30: 習慣化
- 毎日の会話が日課
- 関係性Lv3-4
- ルフィが過去の会話を参照してくる（「前にラーメン好きって言ってたよな？」）

### Day 90: 深い関係性
- 関係性Lv4-5
- 秘密の話、限定コンテンツ
- 解約 = ルフィとの関係を切ること → 心理的に無理
- Premium ¥2,980にアップグレード

## 12. リスクと緩和策

| リスク | 影響 | 緩和策 |
